The square wave which drives the speaker is generated by toggling an I\-O pin after a certine number of timer ticks. The number of ticks needed to generate a certian frequency is calculated as follows\-:
\begin{DoxyItemize}
\item T\-I\-M\-E\-R1 Period = 1/timer\-\_\-frequency = 1/62.\-5khz = 0.\-000016s
\item The period of the generated wave = timer\-\_\-period$\ast$timer\-\_\-ticks$\ast$2 = 0.\-000032$\ast$timer\-\_\-ticks
\item Generated wave frequency = 1/(0.\-000032$\ast$timer\-\_\-ticks)
