DESCRIPTION

Library for interfacing to a quadrature encoder. An interrupt is used to read the encoder increments. The interrupt declaration is #defined as ROTARY_ENCODER_ISR. To use this library, you need to include ROTARY_ENCODER_ISR in the main file of your orogram.

HARDWARE

Target Device: ATTINY26L

Hardware Description: By default a stepper motor is connected to PORTA pins 4-7 as follows: PA4: Disable2; PA5: Phase2; PA6: Disable1; PA7: Phase1 This can be changed using the #defines below
