DESCRIPTION

This program implements a servo motor controller. It detects a motor's position using a potentiometer and uses a proportional-derivative control system to control the motor's speed in order to move the motor to its target position.

HARDWARE

Target Device: ATTINY26L

Macro Definition Documentation

#define F_CPU   1000000UL

The system clock frequency is defined as specified in the tiny26 datasheet.

The default clock frequency for the tiny26 is 1MHz. This define is used by the _ms_delay() function in delay.h to introduce accurate delays.
