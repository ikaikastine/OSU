Detailed Description

Contains the definition for the background thread that polls the ADC input from the solar panel and sends that value to the GUI to be displayed to the user.

def tiny26.SolarTrackingThread.run (   self)

Called when the thread is started.

As long as the stop flag is not set, the thread sends a request to the Tiny for solar measurements. When the computer receives a response from the Tiny, it fires a SolarEvent, which tells the GUI to update itself with the latest solar measurements.
